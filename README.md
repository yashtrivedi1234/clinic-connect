# MediCare Clinic

Expert vertigo and neurology care.

## Getting Started

Follow these steps to run the project locally:

1.  **Clone the repository**
2.  **Install dependencies**: `npm install`
3.  **Start the development server**: `npm run dev`

## Technologies

-   Vite
-   TypeScript
-   React
-   shadcn-ui
-   Tailwind CSS
